### わかったこと

- インタフェース型は、**指定したメソッドを持つ型であれば代入できる**

- Goではエラーはインタフェースとして定義されている
  - つまり、Errorメソッドがある構造体は全てエラーとして扱える

    ```go
    type error interface {
        Error() string
    }
    ```
